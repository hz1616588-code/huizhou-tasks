# 匯洲國際工單系統

> 中壢（客服）⇄ 龍潭（倉管）雙向工單協作 + 即時彈窗通知

## 系統架構

| 頁面 | 用途 | 對象 |
|---|---|---|
| `cs.html` | 客服端 | 中壢客服（元寶）、龍潭客服（純純） |
| `warehouse.html` | 倉管端 | 龍潭倉管（小梁、倉管大哥） |
| `admin.html` | 管理員 | 小瀾 |

## 技術棧

- 純 HTML + CSS + JS（無框架）
- Supabase（資料庫 + Realtime + 檔案儲存）
- GitHub Pages（部署）

## 角色權限

| 功能 | 客服 | 倉管 | 管理員 |
|---|---|---|---|
| 開工單 | ✅ | - | ✅（代開） |
| 看所有工單 | ✅（含對方部門） | ✅ | ✅ |
| 完成工單 | - | ✅ | ✅（代完成） |
| 作廢工單 | ✅（限自己開的且未完成） | - | ✅（任何 pending） |
| 編輯工單 | - | - | ✅ |
| 人員管理 | - | - | ✅ |
| 收新工單通知 | - | ✅ | ✅ |
| 收完成通知 | ✅ | - | ✅ |

## 通知機制

新工單 / 完成事件發生時：
- 🔔 右下角 Toast 彈窗（10 秒淡出，可手動關）
- 🔊 警報音（急件 5 聲、一般 3 聲）
- 📢 標題列閃爍（切回分頁自動停）
- 💻 Windows 桌面通知（需允許權限）
- 📋 通知清單累積（雲端同步，最近 7 天）

## 資料保留

- 處理中工單：永不刪除
- 已完成 / 已作廢工單：保留 3 個月後自動清理（每天凌晨 3 點）
